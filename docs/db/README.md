# スキーマ設計

## テーブル名

`spbdb`

## テーブル定義

各テーブルの構成です。

### member_role

役職などを管理するテーブルです。

| カラム名   | column_name     | 型         | PK  | NULL 許可 | 概要                 |
| ---------- | --------------- | ---------- | --- | --------- | -------------------- |
| ID         | id              | INT        | o   | x         | テーブル管理用 ID    |
| 役職名     | role_name       | VARCHAR(8) | x   | x         | 役職の名称           |
| 重みづけ値 | weighting_value | INT        | x   | x         | 重みづけ計算に使う値 |
